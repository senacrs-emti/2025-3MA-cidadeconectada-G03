# 2025-3MA-cidadeconectada-G03

ColetaJA é uma plataforma web responsiva que exibe um mapa interativo com a rota do caminhão de coleta, horários previstos de passagem e atualização em tempo real da localização do veículo. O projeto utiliza JavaScript, HTML, CSS, PHP e integra também um banco de dados para armazenar e gerenciar informações essenciais do sistema.
